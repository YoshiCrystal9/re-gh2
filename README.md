# re-gh2 - Guitar Hero II Recompiled

> [!IMPORTANT]
> This is a proof of concept and uses a version of rexglue that may break this in the future due to it being still in development.
> Also performance isn't the best and is prone to crashes.

Recompiled using the [ReXGlue SDK](https://github.com/rexglue/rexglue-sdk)

# Setup

1. Ensure REXSDK is set properly in your environment - see [ReXGlue SDK](https://github.com/rexglue/rexglue-sdk) for more info
1. Extract a Guitar Hero II iso into `assets`
1. Run cmake configure
1. Run gh2test_codegen utility target from cmake
1. Run cmake build for gh2test
1. Launch with first argument pointing to the assets folder
